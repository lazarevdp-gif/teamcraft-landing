# Эффективная команда — Лендинг курса по командообразованию

Современный лендинг для продажи интенсива по командообразованию для руководителей и собственников бизнеса.

## Решение задачи

Большинство компаний теряют **20–30% маржи** из-за разрозненных отделов, микроменеджмента и отсутствия единой системы коммуникации. Руководители тратят часы на тушение пожаров вместо стратегии, сотрудники выгорают, а бизнес теряет клиентов.

Этот лендинг решает задачу **презентации и продажи курса**, который обучает руководителей:

- Выстраивать кросс-функциональные регламенты и SLA между отделами
- Настраивать систему делегирования и матрицу принятия решений
- Проводить каскадирование целей и создавать командные договоры

Формат: 3 стратегических лекции + 2 практические сессии с реальной командой клиента + готовые шаблоны для внедрения.

---

## Для кого (Целевой пользователь)

- **Собственники бизнеса**, которые устали от микроменеджмента и срывов дедлайнов
- **Руководителиdepartments**, которые тратят 4+ часа в день на согласования и разбор полётов
- **Топ-менеджеры**, чьи стратегии игнорируются на местах
- Команды от **5 до 15 человек**, где отделы работают «как лебедь, рак и щука»

## Технологии

- **HTML5** — семантичная верстка
- **CSS3** — CSS-переменные, glassmorphism, анимации
- **Vanilla JavaScript** — без фреймворков и зависимостей
- **Google Fonts** — шрифт Montserrat

### Эффекты и анимации

- Glassmorphism-карточки (`backdrop-filter: blur`)
- Scroll-анимации через `Intersection Observer` (fade-up, slide-left/right, scale)
- Параллакс на фоновых элементах
- Hover-эффекты с трансформациями и свечением кнопок
- Анимация счётчиков статистики
- Бегущая строка (marquee)
- Preloader с анимацией загрузки
- Sticky-header с blur при скролле
- Модальное окно с анимацией появления
- Shimmer-эффект на кнопках при наведении

## Сбор данных (Google Sheets)

Форма записи отправляет данные в Google Таблицу через Google Apps Script.

### Подключение

1. Создайте Google Таблицу с заголовками: `Дата | Имя | Фамилия | E-mail | IP адрес | Источник`
2. Скопируйте ID таблицы из URL
3. Откройте **Расширения → Apps Script**
4. Вставьте код из файла `google-apps-script.js`
5. Вставьте ID таблицы в переменную `SPREADSHEET_ID`
6. **Deploy → New deployment → Web app → Anyone → Deploy**
7. Скопируйте URL и вставьте в `index.html` в переменную `SCRIPT_URL`

### Возможности скрипта

- Запись данных в Google Таблицу с датой и временем
- Валидация обязательных полей и формата e-mail
- Проверка дубликатов по e-mail
- Email-уведомления о новой заявке
- Автоматическое форматирование таблицы
- Тестовая HTML-форма для проверки
